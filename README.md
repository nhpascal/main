# 이동농협 생활·농업 지식창고

이동농협 조합원과 직원을 위한 생활·농업 정보 및 로컬푸드 레시피 웹페이지입니다.

## 실행 방법

이 프로젝트는 순수 HTML/CSS/JS로 작성되어 별도의 설치 과정이 필요 없습니다.

1. `index.html` 파일을 더블 클릭하여 웹 브라우저에서 엽니다.
2. 또는 VS Code의 'Live Server' 확장 프로그램을 사용하여 실행할 수도 있습니다.

## 주요 기능

- **홈**: 주요 카테고리 바로가기 및 인사말
- **생활·농업 미니백과**: 조합원 가입 기준, 혜택, 농업 기초 지식 등 카드형 정보 제공
- **로컬푸드 레시피**: 이동읍 특산물(백옥쌀, 고구마 등)을 활용한 레시피 제공
- **반응형 디자인**: 모바일과 PC 환경 모두에 최적화

## 파일 구조

- `index.html`: 전체 레이아웃 및 구조
- `style.css`: 디자인 및 스타일링 (반응형 포함)
- `script.js`: 데이터(미니백과, 레시피) 및 화면 전환 로직
